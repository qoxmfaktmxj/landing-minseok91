# 랜딩페이지 전면 재설계 — "THE MINSEOK TIMES"

날짜: 2026-06-12
상태: 사용자 검토 대기
구현: Codex에 위임 (별도 구현 계획 문서로 전달)

## 1. 배경과 목표

기존 랜딩페이지(라이트 미니멀 흑백 + 그린 액센트)는 방문자의 관심을 끌지 못함.
웹 에이전시 수준의 임팩트 있는 디자인으로 전면 재구성한다. 기존 design.md /
DESIGN-voltagent.md는 따르지 않는다.

- **목표 독자**: 채용 담당자 / 조직 리더 — "이 사람 뽑고 싶다"는 인상
- **핵심 메시지**: HR 도메인 9년 × AI 네이티브 개발 × 조직 AX 전환 리더십
- **성공 기준**: 첫 화면에서 기억에 남는 강한 인상, 끝까지 스크롤하게 만드는 내러티브,
  연락/GitHub 클릭 유도
- **포지셔닝 변경**: 채용·커머스 도메인은 핵심 역량에서 제외하고 "Lab(실험작)"으로 강등.
  HR + AI + AX 전환만 전면에 배치

## 2. 컨셉

**"THE MINSEOK TIMES"** — 9년차 AX 엔지니어를 단독 보도하는 신문.

페이지 전체가 한 부의 신문이다. 제호, 발행호(VOL.9 = 연차), 브레이킹 뉴스 티커,
1면 헤드라인, 칼럼 기사, 특집 기사, 광고면(구인광고 패러디)까지 신문의 문법을
끝까지 패러디한다.

- **톤**: 위트. 잘 만든 유머는 실력의 시그널. 단 정보(경력·기술·프로젝트)는 정확하게.
- **모션**: 풀 액션 — "인쇄되는 신문"이라는 컨셉 안에서 화려한 인터랙션.

## 3. 디자인 시스템

| 역할 | 값 | 용도 |

| Ink-soft | `#4a5470` | 보조 텍스트 |
| Ink-faint | `#5a6280` | 리드문, 캡션 |
| Vermilion (액센트) | `#ff4f30` | CTA, BREAKING 라벨, 하이라이트, 제호 포인트, 호버 |

### 3.2 타이포그래피

| 역할 | 폰트 | 비고 |
|---|---|---|
| 제호(masthead) | 세리프 — Noto Serif KR / Georgia 폴백 | 초대형, 900 |

| 인용구(pull quote) | 세리프 이탤릭 | |

### 3.3 신문 질감 디테일

- 굵은 상단 룰(2.5px) + 헤어라인(1px) 이중 괘선
- 칼럼 사이 1px 세로 괘선
- 광고 박스: 2px 점선 테두리, 미세 회전(-0.5deg ~ 1deg)
- 섹션 헤더: 가운데 정렬 대문자 + letter-spacing 3px, 상하 괘선
- 날짜줄: "SEOUL, KOREA / 오늘의 커밋: N / FREE 무가지" 같은 위트 메타

## 5. 모션 설계 (풀 액션)

1. **인쇄 인트로** (최초 로드 1회): 제호가 도장 찍히듯 등장 → 괘선이 그어지고 →
   헤드라인 잉크가 번지듯 나타남. 세션당 1회(sessionStorage), 2초 이내, 스킵 가능
2. **헤드라인 글자 단위 등장**: 스탬프/타이핑 느낌의 stagger 애니메이션
3. **BREAKING 티커**: 무한 마퀴 (CSS transform 루프)
4. **커스텀 커서**: 잉크 점 → 인터랙티브 요소 호버 시 "READ"/"OPEN" 도장으로 변형.
   데스크톱 포인터 환경에서만 (`pointer: fine`)
5. **스크롤 패럴랙스**: 칼럼별 스크롤 속도 차, 헤드라인 미세 무브
6. **숫자 카운트업**: viewport 진입 시
7. **특집 기사 호버 프리뷰**: 커서 따라다니는 라이브 스크린샷/프리뷰 카드
8. **긴급구인 광고**: 펄스 배지 + 호버 시 도장 찍히는 효과
9. **스무스 스크롤**: lenis
10. **접근성**: `prefers-reduced-motion` 시 인트로·패럴랙스·마퀴 비활성(정적 폴백),
    커스텀 커서 미적용. 모바일: 커서/패럴랙스 제외, 나머지 유지

## 6. 기술 방침

- 스택 유지: Next.js 16 (App Router) + Tailwind CSS 3 + framer-motion 12 + TypeScript
- 추가 의존성: `lenis` 1개만. GSAP 금지 — framer-motion `useScroll`/`useTransform`으로 구현
- 폰트: Noto Serif KR(제호·인용) 추가 — `next/font`, 나머지 기존 폰트 재사용
- 콘텐츠: `src/data/siteContent.ts` 전면 재작성 (신문 카피 구조로 스키마 변경),
  `projects.ts` 큐레이션 (featured / lab 구분 필드)
- 컴포넌트: `src/components/sections/*` 전부 교체, `newspaper/` 네임스페이스로 신규 작성
- globals.css 토큰 전면 교체 (3.1 팔레트)
- SEO/메타: 타이틀·OG도 신문 컨셉 반영 ("The Minseok Times — HR × AI × AX")
- 성능: 마퀴·패럴랙스는 transform/opacity만 사용, layout 트리거 금지. LCP는 헤드라인
  텍스트 — 인트로 애니메이션이 LCP를 가리지 않도록 텍스트는 즉시 DOM에 존재

## 7. 테스트 / 검증

- `npm run lint` + `npm run build` 통과
- 뷰포트 검증: 375px / 768px / 1440px 스크린샷
- `prefers-reduced-motion` 모드 동작 확인
- 라이트하우스: 성능 90+ 목표 (모션 있어도 CLS 0 유지)
- 카피 오탈자 및 링크(라이브 프로젝트 5개, GitHub, 블로그) 전수 클릭 확인
